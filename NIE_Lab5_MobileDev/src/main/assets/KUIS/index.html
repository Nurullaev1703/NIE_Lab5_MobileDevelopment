Парк Афганцев

ул. Лермонтова Павлодар Казахстан

Парк Афганцев - парк, служащий напоминанием о событиях Афганской войне, где принимали участие Казахстанские военные для устранения конфликта.
В 2018 году парк перенес ряд улучшений и ремонтов. Также здесь открыли аллею памяти о 36 погибших солдатах из Павлодарской области. Для каждого из них был создарн памятник и специальная ваза для цветов в виде гильзы
